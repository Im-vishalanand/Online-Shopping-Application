## Setting & Installation 

Install the Spring Tools Suite 
```bash
https://spring.io/tools
```

```bash
https://dev.mysql.com/downloads/mysql/
```

Clone the Repository

```bash
git clone https://github.com/Im-vishalanand/adhesive-cactus-7535.git
```
